# Deskripsi Program
Program ini dibuat untuk mengimplementasikan algoritma *Selection Sort* menggunakan bahasa pemrograman Java. Selection Sort merupakan salah satu metode pengurutan data yang bekerja dengan cara mencari nilai terkecil dari sebuah array kemudian menukarnya dengan posisi elemen yang sedang diproses.

Pada program ini, proses pengurutan dilakukan secara bertahap melalui beberapa iterasi hingga seluruh data tersusun secara terurut dari nilai terkecil ke nilai terbesar. Program juga menampilkan hasil setiap proses iterasi sehingga pengguna dapat memahami bagaimana algoritma Selection Sort bekerja dalam mengurutkan data.
